Bienvenue sur ExilTool

Pour pouvoir participer au site vous devez installer le plugin Tampermonkey sur votre navigateur.

Ensuite utilisez le lien Script dans le menu.
Tampermonkey devrait détecter le script et vous proposer de l'installer.
Il vous faudra aussi autoriser l'envoi d'information lors de la première utilisation.

Le script permet d'automatiser l'envoie d'informations vers ExilTool. Pour le momment celà comprend :

  • Stats des planètes visibles.
